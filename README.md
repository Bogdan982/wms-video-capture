# WMS Video Capture — Android App

Приложение для видеосъёмки по команде из WMS предприятия.

## Назначение

1. Получает идентификатор заказа из WMS (через Intent)
2. Показывает уведомление: *«Получен идентификатор заказа №ХХХ. Начать съёмку?»*
3. Кнопка «Начать съёмку» с защитой от случайного нажатия (удержание 2 сек)
4. Штатная камера устройства записывает видео
5. После записи — диалог: «Использовать ID {X}, отсканировать QR или ввести вручную?»
6. Видео выгружается на SMB/CIFS-ресурс предприятия (папка с ID задачи)
7. Локальная копия удаляется
8. Файл-флаг отправляется на WMS
9. **Логи** автоматически отправляются на SMB-ресурс предприятия

## Пользовательский интерфейс

Пользователю доступно **только**:
- **Начать съёмку** (с защитой от случайного нажатия)
- **Завершить съёмку**
- **Сканировать QR-код** для замены ID (опционально)
- **Настройки** — по паролю администратора

## Технологии

- **Kivy** — UI-фреймворк
- **pyjnius** — доступ к Android API (камера, Intent)
- **pysmb** — SMB/CIFS-клиент для выгрузки
- **ZXing Barcode Scanner** — QR-сканирование (через Intent)
- **requests** — HTTP-флаг на WMS

## Поток выполнения

```
WMS → Intent(task_id="ABC-123")
  → ConfirmScreen: "Получен ID №ABC-123. Начать съёмку?"
    → Кнопка [удержание 2 сек] → Запись (системная камера)
      → DoneScreen: "Использовать ABC-123 или отсканировать QR?"
        ├── [Использовать ABC-123] → UploadScreen
        └── [Сканировать QR] → ScanScreen → новый ID → UploadScreen
          → SMB: /captures/{ID}/video.mp4
          → Удаление локального
          → HTTP-флаг на WMS
          → DoneScreen: ✅
```

## Защита от случайного нажатия

Кнопка «Начать съёмку» на ConfirmScreen:
- **Удержание 2 секунды** с прогресс-баром
- Преждевременное отпускание сбрасывает прогресс
- После полного удержания — кнопка активируется для нажатия

## Настройки

Настройки защищены **паролем администратора** (по умолчанию: `admin`).

Доступные настройки:
- **SMB**: хост, шара, пользователь, пароль
- **WMS**: базовый URL
- **Логирование**: хост и шара для логов
- **Смена пароля администратора**

## Логирование

Все события приложения логируются:
- Локальный файл с ротацией (512 КБ)
- Автоматическая выгрузка на SMB каждые 30 минут
- Формат: `{дата}_{устройство}.log`

## QR-сканер

Требуется установка **ZXing Barcode Scanner** (бесплатно в Google Play):
- https://play.google.com/store/apps/details?id=com.google.zxing.client.android
